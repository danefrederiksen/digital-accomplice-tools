Methodology: 20 buyer-intent queries were run through Claude (Anthropic) with live web search enabled. Each response was scored 0–3 for each of the 4 companies analyzed (0 = Absent, 1 = Cited in sources only, 2 = Mentioned in text, 3 = Recommended by name). Raw scores were summed and normalized to a 0–10 scale (raw / 60 × 10). YouTube channels were verified via live web search. Video scores were assigned based on channel activity, subscriber count, upload frequency, and content variety. AI answers change over time — these results represent a point-in-time snapshot, not a guarantee.
