P4 Single Source of Truth — Unified Progress Dashboard

4A. The Problem

Right now each tool tracks its own daily progress independently. There is no single place to see how your day is going across all 11 tools. You have to open each tool's tab to check counts, which means you'll miss things.

4B. Proposed Solution: Hub Dashboard

Build a lightweight "Hub" page (port 3849 or similar) that reads all data files and shows:

  • Today's progress bars for each tool (DMs sent, emails sent, comments logged, connections requested)
  • Combined daily totals vs targets (e.g., 45/60 touches today)
  • Weekly totals for the Friday review (comments, outreach, replies, reply rate)
  • Quick-links to open each tool
  • Alerts for overdue follow-ups, stale prospects, tools with no activity today

This is a read-only dashboard — no writes, no race conditions. Just reads all JSON files and aggregates.
