System Overview

The DA Prospecting Tools system is a suite of 12 purpose-built web applications for managing LinkedIn DM outreach, email campaigns, and engagement tracking. All tools are locally hosted Node.js + Express servers with single-file HTML front-ends, organized by prospect segment and outreach channel. A unified Hub Dashboard aggregates data from all 12 tools into a single view.

Hub Dashboard
Unified view of all 12 tools with score cards, alerts, tool cards, weekly summary table, and quick links. Port 3849.
A/B Message Testing
Round-robin variant assignment on 5 active DM tools. Tracks sent/replied per variant with dedicated A/B Report tab.
Template Editor
Editable message templates per sequence step with {name}/{company} variables and copy-to-clipboard.
Follow-Up Engine
Automated follow-up scheduling: offer → follow-up 1 → follow-up 2 → final nudge → exhausted.
Comment Queue
Unified search across 4 tools (B2B 2nd, Cyber 2nd, Referral 1st, Referral 2nd). Logs comments back to source. 8/day target.
One-Click Launch
node start-all.js starts all 12 tools + hub dashboard on ports 3849, 3851–3862.

A/B Testing System

A/B message testing is now live on all 5 active DM tools. When a user copies a message, the system assigns variant A or B via round-robin and persists the assignment to the server. Each tool includes an A/B Report tab that shows per-step sent/replied/rate for each variant.

Feature Description
Variant AssignmentRound-robin (A, B, A, B...) on first copy per step per prospect
PersistenceStored in abVariants field on prospect record, saved server-side via PUT
TemplatesEach step has _a and _b template variants, editable in Template Editor
ReportingA/B Report tab shows: step name, variant, sent count, replied count, reply rate %
Visual IndicatorOrange "A" or blue "B" badge on each prospect card after assignment
