# AI Snapshot — Quick Reference Checklist

Print this. Use it every time.

---

## BEFORE YOU START
- [ ] Company name + website confirmed
- [ ] 3 competitors confirmed WITH DANE (never assume)
- [ ] Mode selected: **Lite** (60 min, cold) or **Full** (3.5 hr, warm)
- [ ] Tracking sheet created (copy template)
- [ ] Screenshot folder created in Google Drive

## PROMPT GENERATION (10 min)
- [ ] Customize prompt templates for this industry
- [ ] Lite: 10 prompts (Categories 1-4)
- [ ] Full: 15 prompts (Categories 1-5) + 5 video prompts
- [ ] Prompts saved in tracking sheet Tab 1

## QUERY RUNS
### Lite Mode
- [ ] Run 1 — ChatGPT: all prompts, new session each, screenshot each
- [ ] Run 1 — Perplexity: all prompts, new session each, screenshot each
- [ ] Record all data in tracking sheet Tab 2

### Full Mode
- [ ] Run 1 (Day 1) — all 4 models, all prompts
- [ ] Run 2 (Day 2) — all 4 models, all prompts
- [ ] Run 3 (Day 3) — all 4 models, all prompts
- [ ] Record all data in tracking sheet Tab 2

## SCORING (15 min)
- [ ] Score each query: 3=Recommended, 2=Mentioned, 1=Cited, 0=Absent
- [ ] Calculate per-model normalized scores (raw/max × 10)
- [ ] Calculate overall score (average of model scores)
- [ ] Full mode: calculate consistency % (Tab 4)
- [ ] Full mode: calculate cross-model agreement (Tab 5)

## VIDEO RESEARCH (20 min)
- [ ] YouTube channel data for all 4 companies (Tab 6)
- [ ] Website video check for all 4 companies (Tab 6)
- [ ] Note which companies had video cited in AI results

## ACCURACY CHECK (10 min)
- [ ] AI correctly describes target company? 
- [ ] Any hallucinated products/features/pricing?
- [ ] Perplexity citations — URLs valid and recent?
- [ ] Flag any inaccuracies for the report

## REPORT ASSEMBLY (30 min)
- [ ] Open DA report template
- [ ] Fill cover page
- [ ] Write "The Bottom Line" (1-2 sentences)
- [ ] Fill scorecard with color-coded scores
- [ ] Fill AI visibility tables
- [ ] Fill video comparison tables
- [ ] Write 3 specific gaps with evidence
- [ ] Write 3 recommendations with timelines + investment ranges
- [ ] Add Calendly link
- [ ] Fill appendix

## QUALITY CHECKS
- [ ] 60-second test: can someone get the point instantly?
- [ ] Every claim backed by data?
- [ ] Screenshots archived?
- [ ] Non-determinism acknowledged?
- [ ] Would a marketing manager forward this to their CEO?
- [ ] Calendly link works?

## SEND IT
- [ ] Lite: email directly or attach to LinkedIn DM
- [ ] Full: email with 2 questions + Calendly link
- [ ] Log in prospect warming tracker
- [ ] Schedule LinkedIn follow-up for 2 days later
